  • Design Process

    Creating a clear blueprint for the website is critical. The design process works much smoother if details are hashed out before coding is started.

    In the past traditionally layout the details in a familiar Adobe program. Balsalmiq is a great tool I have found that allows the designer to create rough web templates from drag and drop resizable elements.

    The goals of this site are to showcase my letterpress work and also to educate people interested in learning the letterpress process. Having a website that adapts to different screen sizes, especially a phone is crucial these days. This website would be responsive to user screen sizes and one that I could easily update since I have new work to display every few weeks.

  • Design Process

    There are multiple javascript libraries which allow designers to pull complex components into their site and tweak them as needed. Jquery is probably the most well known library which I have used to add interactivity into the site. Twitter Bootstrap, pictured here, was used throughout the site to add complex elements through Jquery. CSS styles were then modified to adjust appearance.

  • Design Process

    While Javascript libraries help to incorporate complex elements into websites with little code, sometimes additional coding is required for more specific functionally. The sortable gallery functions on javascript controls and allows for the user to filter by assigned data-attributes.

  • Design Process

    After the planning and coding phases, the website is is tested on all major browsers for inconsistancies which almost always tend to crop up. Once everything is sorted out the site is ready to launch!
